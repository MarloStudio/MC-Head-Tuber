# MC-Head-Tuber
A 3d model of any minecraft skin that has head and mouth tracking, opening the mouth when you talk and looking where you look.
------------------------------------------------------------------------------------------------------------------------------
Made by Marlo with the help of Claude AI and Google Gemini
------------------------------------------------------------------------------------------------------------------------------

Use the sliders to mess with the head until you like how it looks
Any minecraft skin is supported (as long as it's usable in vanilla)
The head will start off kinda funky, so just use the recalibrate button

To use in obs, just use the window capture tool, right click on the tool, hit effects, then chroma key. After that, just crop and resize to taste
